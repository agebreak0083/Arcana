=================================================================
전투 UI 설정 가이드
=================================================================

이 가이드는 전투 화면에 라운드/턴 정보와 스킬 이름을 표시하는 UI를 설정하는 방법을 설명합니다.

=================================================================
1. Canvas 생성
=================================================================

1) Hierarchy 창에서 우클릭 → UI → Canvas
2) Canvas 이름을 "BattleCanvas"로 변경
3) Canvas 설정:
   - Render Mode: Screen Space - Overlay
   - Canvas Scaler 설정 (선택사항):
     • UI Scale Mode: Scale With Screen Size
     • Reference Resolution: 1920 x 1080

=================================================================
2. 라운드/턴 텍스트 생성 (상단)
=================================================================

1) BattleCanvas 우클릭 → UI → Text - TextMeshPro
   (처음 사용 시 Import TMP Essentials 클릭)

2) 이름을 "RoundTurnText"로 변경

3) Rect Transform 설정:
   - Anchors: Top Center (상단 중앙)
     • Anchor Presets 창에서 Shift + Alt 누른 채 Top Center 클릭
   - Pos Y: -50 (상단에서 50픽셀 아래)
   - Width: 400
   - Height: 80

4) TextMeshProUGUI 컴포넌트 설정:
   - Text: "Round 1 - Turn 1"
   - Font Size: 40
   - Alignment: Center (가로), Middle (세로)
   - Color: 흰색 또는 원하는 색상
   - Font Style: Bold (선택사항)

5) Outline 추가 (선택사항, 가독성 향상):
   - Add Component → UI → Effects → Outline
   - Effect Color: 검은색
   - Effect Distance: (2, -2)

=================================================================
3. 스킬 이름 텍스트 생성 (하단)
=================================================================

1) BattleCanvas 우클릭 → UI → Text - TextMeshPro

2) 이름을 "SkillNameText"로 변경

3) Rect Transform 설정:
   - Anchors: Bottom Center (하단 중앙)
     • Anchor Presets 창에서 Shift + Alt 누른 채 Bottom Center 클릭
   - Pos Y: 100 (하단에서 100픽셀 위)
   - Width: 600
   - Height: 100

4) TextMeshProUGUI 컴포넌트 설정:
   - Text: "가드 슬래시"
   - Font Size: 48
   - Alignment: Center (가로), Middle (세로)
   - Color: 노란색 또는 강조색
   - Font Style: Bold

5) 배경 패널 추가 (선택사항):
   - SkillNameText 우클릭 → UI → Image
   - 이름을 "Background"로 변경
   - Hierarchy에서 드래그하여 SkillNameText의 자식으로 만들기
   - Rect Transform에서 Anchor Presets의 Stretch (우하단) 선택
   - Left/Right/Top/Bottom 모두 0으로 설정
   - Image 컴포넌트:
     • Color: 반투명 검은색 (R:0, G:0, B:0, A:180)
   - Hierarchy에서 Background를 SkillNameText 위로 드래그하여 뒤로 보내기

6) Outline 추가 (선택사항):
   - Add Component → UI → Effects → Outline
   - Effect Color: 검은색
   - Effect Distance: (3, -3)

=================================================================
4. BattleUI 스크립트 연결
=================================================================

1) BattleCanvas 선택

2) Add Component → Scripts → BattleUI

3) BattleUI 컴포넌트 설정:
   - Round Turn Text: RoundTurnText 드래그 앤 드롭
   - Skill Name Text: SkillNameText 드래그 앤 드롭
   - Skill Name Display Time: 2 (2초간 표시)

=================================================================
5. 초기 상태 설정
=================================================================

1) SkillNameText는 스크립트에서 자동으로 숨김 처리됩니다.
2) 게임 시작 시 "Round 1 - Turn 1"이 자동으로 표시됩니다.
3) 스킬 사용 시 하단에 스킬 이름이 표시되고 2초 후 사라집니다.

=================================================================
6. 커스터마이징 (선택사항)
=================================================================

■ 폰트 변경:
   - 원하는 TrueType Font (.ttf) 파일을 임포트
   - Window → TextMeshPro → Font Asset Creator로 폰트 생성
   - TextMeshProUGUI의 Font Asset에 적용

■ 애니메이션 추가:
   - Window → Animation → Animation
   - SkillNameText 선택 후 Create 클릭
   - FadeIn/FadeOut, Scale, Slide 등의 애니메이션 추가
   - BattleUI.cs의 ShowSkillName()에서 애니메이션 재생

■ 레이아웃 변경:
   - RoundTurnText를 좌측 상단으로 이동
   - SkillNameText를 중앙으로 이동
   - 추가 정보 표시 (HP, MP, 턴 수 등)

=================================================================
7. 테스트
=================================================================

1) Play 모드로 진입
2) 전투가 시작되면 상단에 "Round 1 - Turn 1" 표시 확인
3) 캐릭터가 스킬을 사용하면:
   - 턴이 증가하는지 확인
   - 하단에 스킬 이름이 표시되는지 확인
   - 2초 후 스킬 이름이 사라지는지 확인

=================================================================
8. 문제 해결
=================================================================

■ UI가 보이지 않을 때:
   - Canvas의 Render Mode가 Screen Space - Overlay인지 확인
   - Camera가 UI를 가리고 있는지 확인
   - Text Color의 Alpha 값이 255인지 확인

■ 텍스트가 잘릴 때:
   - Rect Transform의 Width/Height를 늘리기
   - Auto Size 옵션 활성화 (TextMeshProUGUI)

■ 스킬 이름이 업데이트되지 않을 때:
   - BattleUI 스크립트가 BattleCanvas에 붙어있는지 확인
   - BattleUI의 참조가 올바르게 연결되었는지 확인
   - Console 창에서 에러 메시지 확인

■ BattleUI.Instance가 null일 때:
   - Scene에 BattleUI 스크립트가 하나만 있는지 확인
   - BattleUI가 BattleManager보다 먼저 실행되도록 설정
